Neubrutalist Components

Colors

Any component with a color attribute accepts any of the following values: white (usually default), red, orange, yellow, green, teal, blue, purple, or magenta.

For each color, there are three related CSS variables (light, medium, dark) of varying saturations and lightness values.

<neu-avatar>

  • alt: supplies the underlying image's alt value
  • color: determines the color theme
  • round: rounds the container and image to circles
  • size: defines the height and width of the displayed image
  • src: the URL for the contained image

<neu-button>

  • color: determines the color theme
  • round: rounds the button to an ellipse/"pill" shape
Default button Round yellow button

<neu-checkbox>

  • checked: sets the state to checked
  • color: determines the color theme (default "white")
  • label: labels the control

<neu-input>

  • label: labels the control
  • max: minimum value for a number control
  • min: minimum value for a number control
  • placeholder: rounds the button to an ellipse/"pill" shape
  • step: incremental value for a number control
  • type: sets the underlying field type (default "text")

<neu-select>

  • options: a comma-separated list of values
  • placeholder: sets the empty placeholder text

<neu-slider>

  • color: determines the color theme
  • label: labels the control
  • max: maximum value
  • min: minimum value
  • step: incremental value

<neu-textarea>

  • label: labels the control
  • placeholder: sets the underlying field's empty placeholder text

<neu-toggle>

  • color: determines the color theme
  • label: labels the control
